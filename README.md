# ⚔️ ARENA — кооперативный браузерный слэшер (1–4 игрока)

Полноценная многопользовательская игра на **Three.js** с **авторитарным сервером** на Node.js.
Одиночный режим = кооп с одним игроком. Подключение по **Wi-Fi / локальной сети**, с заделом на выход в интернет.

---

## 🚀 Быстрый старт

```bash
# 1. установить зависимости
npm install

# 2. запустить сервер + клиент (dev-режим)
npm run dev
```

После запуска в консоли сервера появятся адреса:

```
🎮  ARENA COOP — игровой сервер запущен
─────────────────────────────────────────────
Локально:      http://localhost:3001
По сети (LAN): http://192.168.1.42:3001   ← раздай этот адрес
─────────────────────────────────────────────
```

- **На этом компьютере:** открой `http://localhost:5173`
- **На телефоне/другом ПК по Wi-Fi:** открой `http://<LAN-IP>:5173`
  (LAN-IP сервер печатает в консоли; клиент и сервер должны быть в одной сети)

> Клиент сам подключается к нужному серверу через `location.host` — один и тот же
> код работает и на localhost, и по LAN, и позже через интернет без правок.

### Как играть компанией
1. Первый игрок жмёт **«СОЗДАТЬ ИГРУ»** → получает **код комнаты** (напр. `K7M2Q`).
2. Остальные вводят этот код в поле **«ВОЙТИ»**.
3. Хост жмёт **«⚔️ НАЧАТЬ БОЙ»**.

## 🎮 Управление

| Действие | Телефон | Компьютер |
|----------|---------|-----------|
| Движение | левый стик | `WASD` |
| Камера | свайп по экрану | зажать `ПКМ` + мышь |
| Атака (комбо ×3) | кнопка ⚔️ | `ЛКМ` |
| Прыжок (двойной) | ⬆️ | `Space` |
| Рывок (неуязвимость) | 💨 | `Shift` |
| Скил 1 | кнопка на панели | `1` |
| Скил 2 | кнопка на панели | `2` |
| Ульта | ⚡ | `F` |
| Инвентарь | — | `I` |
| Гримуар | — | `K` |

Режим определяется автоматически (тач-устройство → телефон, иначе → ПК).

### Сетевая модель
- **Сервер авторитарен**: ИИ врагов, урон, волны, дроп считаются только на сервере.
- **Клиент шлёт намерения** (инпут ~30 Гц): движение, yaw/pitch, граничные действия (прыжок/рывок/удар/ульта).
- **Сервер шлёт снапшоты** мира 20 Гц + мгновенные **события** (урон, убийства, волны).
- **Предсказание**: локальный игрок просчитывается клиентом мгновенно, затем сверяется со снапшотом (reconciliation).
- **Интерполяция**: чужие игроки и враги плавно интерполируются между снапшотами.
- **Античит**: клиент не может сам нанести урон — сервер проверяет дистанцию и конус атаки.

## 🌐 Выход в интернет (на будущее)

Код уже готов: клиент коннектится к `location.host` по `ws://`/`wss://`. Варианты:

1. **Проброс портов** на роутере (3001 + 5173/80) → играть по внешнему IP.
2. **Обратный прокси** (nginx/Caddy) с TLS → `wss://` + `https://`.
3. **Облако**: `npm run build` → `npm start` (сервер раздаёт `dist/` и WS на одном порту).
   Задеплоить на любой VPS / Render / Fly.io, в Caddy настроить HTTPS — и готово.

Для продакшена:
```bash
npm run build    # собрать клиент в dist/
npm start        # сервер раздаёт игру на :3001 (HTTP + WS вместе)
```
